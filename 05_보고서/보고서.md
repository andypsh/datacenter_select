# AI 데이터센터 최적입지 분석 — 지역균형 발전 관점

**제14회 산업통상자원부 공공데이터 활용 아이디어 공모전 출품작**
**부문: 제품·서비스 개발**
**🚀 라이브 데모**: https://datacenter-select-ds9x.vercel.app

## 요약 (Executive Summary)

AI 산업 폭증으로 데이터센터 수요가 급증하고 있으나, 국내 데이터센터의 다수가 수도권에 집중되어 전력계통 부담과 지역 불균형이 심화되고 있다. 본 연구는 산업통상부 산하 공공데이터(KEPCO 전력 인프라·KICOX 산업단지·KEA 재생에너지) 및 보완 공공데이터를 활용하여 전국 109개 시군 단위로 **9개 평가지표**를 종합한 데이터센터 입지 적합도 모델을 구축하였다. 4개의 V2 신규 지표(전력 인프라·산업 클러스터·지역활력·재생에너지)는 광케이블 백본 분포·기존 데이터센터 분포·인구감소지역 지정 현황·재생에너지 허브 거리 등 원본 raw 데이터로부터 **파생변수(derived variables)** 로 직접 산출하였다. 단순 Top-N 후보지 도출을 넘어 **사용자가 직접 가중치를 조절할 수 있는 인터랙티브 의사결정 지원 대시보드**를 제공하며, 가중치 ±20% 민감도 분석으로 결과 견고성을 입증하였다. 분석 결과로부터 5가지 정책 제언을 자동 도출한다.

## 1. 배경 및 문제 정의

### 1.1 국내 데이터센터 산업 현황
- 국내 운영 데이터센터의 수도권 집중도가 높아 전력 송전망 부담 가중
- AI/생성형 모델 수요로 GPU 데이터센터 수요 폭증 추세
- 전력거래소(KPX) 자료에 따른 수도권 송전망 여유율 한계 임박

### 1.2 정책 환경
- **산업통상부**: AI 산업 육성 + 지역균형 발전 동시 추진
- **행정안전부**: 89개 인구감소지역 지정 (2021~)
- **글로벌**: RE100/CBAM 등 ESG 압박 강화

### 1.3 연구 목표
1. 산업부 산하 공공데이터 기반 **9개 지표** 입지 평가 모델 구축
2. **지역활력 기여도** 차원을 평가지표에 정식 통합
3. 정책 담당자용 **인터랙티브 의사결정 지원 도구** 제공
4. 분석 결과 기반 **자동 정책 제언** 도출

### 2.2 공간 단위
- 전국 시군 109개 (특별시·광역시·도내 시·군)
- 시청/군청 좌표 매핑 (`03_분석/coords.py`)

### 2.3 원본 raw 데이터 → 파생변수 (Derived Variables)

본 연구의 핵심 기여는 **V2 4개 신규 지표를 원본 raw 데이터로부터 직접 산출**한 점이다. 산출 레시피는 `03_분석/derive_indicators.py` 에서 재현 가능.

| 파생변수 | 원본 raw 데이터 | 산출 방법 |
|---|---|---|
| `power` (전력 인프라) | `광케이블.xlsx` (66 백본 노드) | 각 시군 좌표 기준 30km 내 광케이블 노드 수 → Min-Max 정규화 |
| `industry` (산업 클러스터) | `데이터센터위치.csv` (89개) | 각 시군 좌표 기준 30km 내 기존 DC 수 → Min-Max 정규화 |
| `vitality` (지역활력) | 행안부 지정 + DC 분포 | `(1 - DC밀도) × 0.6 + 인구감소지역 × 0.4` |
| `renewable` (재생E 접근성) | 호남 태양광/강원 풍력 좌표 | `exp(-dist/100km)` 지수감쇠 → Min-Max 정규화 |

거리 계산은 Haversine 공식 사용. 산출 결과 분포 (109 시군 평균):
- power: 0.304 / industry: 0.194 / vitality: 0.623 / renewable: 0.286

## 3. 방법론

### 3.1 정규화 (Min-Max)
각 지표를 [0, 1] 범위로 정규화. "낮을수록 좋음" 지표(기온·재해·토지비)는 `1 - x`로 반전.

### 3.2 가중합 스코어링
```
종합점수_i = Σ (w_k × normalized_k_i) / Σw_k
```

### 3.3 인터랙티브 가중치 (대시보드)
사용자가 9개 슬라이더로 가중치를 조절하면 즉시 재계산. 3종 시나리오 프리셋 제공:

| 프리셋 | 강조 지표 | 관점 |
|---|---|---|
| 비용 최우선 | 토지비, 산업 집적 | 사업자 ROI |
| 안정성 최우선 | 전력, 재해, 냉각 | 미션 크리티컬 |
| **지역균형 최우선** ★ | 지역활력, 재생에너지 | **14회 공모전 주제 정합** |

### 3.4 민감도 분석
9개 지표 각각의 가중치를 ±20% 변동시켜 **18개 시나리오** 생성. 각 시나리오에서 Top 10 재계산 → 모든 시나리오에 일관되게 등장하는 시군 = "견고한 후보지" (`03_분석/sensitivity_analysis.py`).

### 3.5 비수도권 분석
- 전국 Top 10 vs **비수도권 Top 10** 별도 산출
- 인구감소지역 지정 후보지 식별

## 5. 정책 제언 (대시보드 자동 생성)

대시보드 `정책` 탭에서 현재 가중치 기준으로 자동 생성되는 5가지 제언:

### 5.1 수도권 데이터센터 신규 허가 총량제
- **근거**: 기본 가중치에서도 Top 10 중 수도권 50%, 수도권 송전망 여유율 한계
- **방안**: 권역별 총량제 도입, 비수도권 후보지(음성·괴산) 우선 인허가
- **대상**: 산업통상부 · 국토교통부

### 5.2 한전 송전망 확충 우선순위 = 비수도권 Top 후보지
- **근거**: KEPCO/KPX 데이터 기반 광케이블 백본 인접 권역
- **방안**: 음성·괴산·서천 권역에 154kV 변전소·송전선로 5년 내 확충
- **대상**: 한국전력공사 · 전력거래소

### 5.3 인구감소지역 데이터센터 입주 인센티브 패키지
- **근거**: 비수도권 Top 10 중 9개가 인구감소지역 (=즉시 적용 대상)
- **방안**: (1) 법인세 3년 100% 감면 (2) 전기요금 산업용(을) 적용 (3) 토지비 50% 보조
- **대상**: 행정안전부 · 기획재정부 · 광역 지자체

### 5.4 RE100 대응 — 재생에너지 인접 데이터센터 클러스터
- **근거**: 지역균형 시나리오 Top 5가 모두 호남 (신안·정읍·서천·부안·장성)
- **방안**: 호남(태양광)·강원(풍력) "RE100 데이터센터 특구" 지정 + 발전소 직접 PPA 허용
- **대상**: 산업통상자원부 · 한국에너지공단

### 5.5 데이터센터 폐열 활용 의무화
- **근거**: 비수도권 인구감소지역 입지일수록 인근 농어촌 활용 가능성↑
- **방안**: MW급 데이터센터는 폐열의 50% 이상을 지역난방·스마트팜으로 환원 의무화
- **대상**: 산업통상자원부 · 환경부

3. **추가 지표**: 통신 광케이블 거리·물류 접근성·인력 공급 통계
