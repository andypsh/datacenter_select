# 데이터센터 최적입지 선정 프로젝트 — V2

V1 (2022) 의 분석을 현재 시점 기준으로 다시 짜는 리뉴얼 버전.

## V1 vs V2

- **V1**: `데이터센터_프로젝트` (2022 원본), `데이터센터_프로젝트_정리` (주제별 정리), `데이터센터_최적입지선정_행안부` (2023 행안부 공모전 결과물) — 그대로 보존.
- **V2**: 본 저장소. V1 `데이터센터_프로젝트_정리` 구조를 출발점으로 사용.

## 폴더 구조 (V1에서 가져옴)

```
01_데이터센터/              # 시군구 법정동 번지 코드, 데이터센터 위치, 지번별 에너지
02_계약종별전력사용량/        # 2012~2022 연도별 일반전력
03_태풍지진/                # 태풍·지진 전처리 + 가중치
04_sw기업개수/              # SW 기업 분포, 광케이블 지도(shp)
05_기온/                   # AWS 관측지점, 평균기온, 재해위험도
06_실거래가/                # 실거래가 + 가중치 데이터
07_QGIS병합/               # QGIS 병합 결과물
기타_자료/                  # 참고 PDF / 시장 현황
발표자료/                   # 보고서, 발표 PDF/PPTX
```
