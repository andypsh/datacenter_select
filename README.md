## 폴더 구조

```
공모전/
├── 00_공모전정보/         # 14회 요강, 평가기준
├── 01_프로젝트기획/       # 컨셉, 차별화 포인트, 산출물 사양 (Vue 확정)
├── 02_데이터/
│   ├── raw/              # 원본 공공데이터 (한전/전력거래소/산단공/기상청 등)
│   ├── processed/        # 전처리 완료 데이터
│   └── 출처매핑.md         # 9개 지표 ↔ 산업부 산하 기관 데이터 매핑
├── 03_분석/              # Python 분석 (모던 pandas/geopandas)
│   ├── coords.py          # 시군 위경도 매핑 (109개)
│   ├── build_scores.py    # CSV → regions.json
│   └── requirements.txt
├── 04_시각화/
│   └── frontend/          # Vue 3 + Vite + TypeScript SPA
├── 05_보고서/             # 최종 보고서 PDF (배경→방법→결과→정책)
├── 06_발표/               # PPT, 데모 영상
└── _v1_data/             # 2022년 V1 분석 자료 (참조용)
```

---

## 빠른 시작

```bash
# 1) 분석: V1 CSV → JSON 데이터 빌드
cd 03_분석
pip install -r requirements.txt
python build_scores.py
# → 04_시각화/frontend/public/data/regions.json 생성

# 2) 프론트엔드 dev 서버
cd ../04_시각화/frontend
npm install
npm run dev
# → http://localhost:5173
```
