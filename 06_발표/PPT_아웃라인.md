## 1. 표지 (1장)
- 제목: **AI 데이터센터 최적입지 — 지역균형 발전 관점**
- 부제: 산업통상부 산하 공공데이터 기반 9개 지표 의사결정 지원 도구
- 팀명 / 발표자명 / 날짜

## 2. 문제 정의 (1장)
- 한국 데이터센터 **수도권 60% 집중** (큰 숫자, 비주얼)

## 3. 왜 지금인가? (1장)

- 산업부 정책: AI 산업 육성 + 지역균형 발전 (★공모전 주제와 정합)
- RE100/CBAM 글로벌 압박

## 4. 우리의 접근 (1장)
- **단순 입지 분석이 아닌 정책 의사결정 지원 도구**
- 사용자가 직접 가중치 조절 → 즉시 시각화
- 9개 지표 / 시나리오 3종 / 정책 제언 5종

## 5. 데이터: 산업부 산하 데이터 명시 (1장)
- 9개 지표 → 출처 표

- 보완: 기상청, 국토부, 통계청

## 6. 9개 평가지표 (1장)
- 카테고리별 색상 코딩 (전력·안전·비용·산업·**지역활력**·ESG)
- 14회 주제 키워드 매칭 (지역활력 = 인구감소지수+GRDP)

## 7. 방법론 (1장)
- Min-Max 정규화 → 가중합 스코어링
- 인터랙티브 가중치 → 즉시 재계산
- 다이어그램: 데이터 → 정규화 → 가중치 → 점수 → Top 10

## 8. 시나리오 프리셋 3종 (1장)
- 비용 최우선
- 안정성 최우선
- **지역균형 최우선** ★ (공모전 주제 정합)
