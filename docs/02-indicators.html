029개 평가지표 상세

전체 9개. 각 지표의 의미·방향·가중치·산출 방법·구현 상태.

전체 요약표

#지표카테고리방향기본 가중치상태
1전력 안정성전력↑ 좋음18%실측
2자연재해 위험안전↓ 좋음10%V1
3평균기온환경↓ 좋음8%V1
4토지 비용비용↓ 좋음10%V1
5산업 집적도산업↑ 좋음10%실측
6통신 인프라통신↑ 좋음9%실측
7IT 인력인력↑ 좋음8%실측
8지역활력 ★지역활력↑ 좋음18%행안부 공식
9재생에너지ESG↑ 좋음9%실측

★ 지역활력은 14회 공모전 주제 직격 지표 — 가중치 18%로 최상위 (전력 안정성과 동률).

1. 전력 안정성 (power_stability)

의미: 데이터센터 운영의 핵심 — 변전소·송전망 인프라가 충분한 시군일수록 안정적.

방향: 높을수록 좋음. 가중치: 18%.

산출:

  • 입력: V1 2012-2022전국.csv (55,132행, 시군구·연월·계약구분·사용량 kWh)
  • 광역시 자치구는 광역시명으로 합산
  • 연간 사용량 시계열에서 3가지 특성 추출:
    • 평균 사용량 (절대 인프라 크기)
    • 5년 추세 (선형 회귀 slope ÷ 평균) — 증가 = 산업 성장
    • 변동성 (CV = stdev ÷ 평균) — 낮을수록 안정
  • 합성: 점수 = 평균×0.5 + 추세×0.3 + (1−변동성)×0.2
  • 109개 시군 min-max 정규화 → 0~100

2. 자연재해 위험 (disaster_risk)

의미: 태풍 + 지진 위험도 평균. 데이터센터는 자연재해에 취약.

방향: 낮을수록 좋음. 가중치: 10%.

산출: V1 가중치데이터_최종.csv의 지진위험도(z) + 태풍위험도(z) 단순 평균.

3. 평균기온 (temp)

의미: 데이터센터 냉각 비용에 직결. 평균기온 낮을수록 자연 냉각 비용 ↓.

방향: 낮을수록 좋음. 가중치: 8%.

산출: V1 가중치데이터_최종.csv의 평균기온(°C) 그대로.

4. 토지 비용 (price)

의미: 부지 확보 초기 투자. 시군별 실거래가 평균 사용.

방향: 낮을수록 좋음. 가중치: 10%.

산출: V1 가중치데이터_최종.csv의 거래금액(만원).

5. 산업 집적도 (companies)

의미: 주변 SW·AI 기업 생태계. 산업 클러스터 효과.

방향: 높을수록 좋음. 가중치: 10%.

산출:

  • V1 SW회사.csv raw 40,722 회사 — 주소 정규식으로 시·도 + 시·군 파싱
  • 광역시 자치구 → 광역시로 합산
  • 기업규모별 가중 카운트: 대=3 / 중견=2 / 중소=1
  • min-max 정규화. UI에는 raw 회사 수도 같이 노출.
왜 가중치를 곱하나: 대기업 본사 1개 = 중소 3개 무게. 이래야 사업 규모·지역 영향력 차이가 반영됨 (단순 카운트는 1인 스타트업 영향 과대평가).

6. 통신 인프라 (telecom_infra)

의미: 데이터센터 ↔ 인터넷 백본 연결성. 광케이블 노드 인접도 proxy.

방향: 높을수록 좋음. 가중치: 9%.

산출:

  • V1 광케이블.csv 66 노드 (지역명 + 위도/경도)
  • 각 시군 중심점 ↔ 모든 노드 haversine 거리 계산
  • 점수 = 50km내 노드×2 + 100km내×1 + 최근접 보너스
  • 109 시군 min-max 정규화

7. IT 인력 (it_workforce)

의미: IT 종사자 공급. 산업집적도와 다른 신호 — 대기업 비중 가중.

방향: 높을수록 좋음. 가중치: 8%.

산출:

  • V1 SW회사.csv raw + 기업규모별 표준 종사자 수 곱
  • 대기업 = 1,500명 / 중견 = 250명 / 중소 = 18명 (한국 IT 업계 평균 추정)
  • 시군별 합계 = 추정 IT 종사자 수
  • log1p 정규화 (서울/판교 압도 완화)
산업 집적도와 다른 시그널: 성남시는 산업집적도 10이지만 IT인력 81 (판교 대기업 비중 ↑). 대전 11→73 (KAIST·연구단지). 두 지표가 진짜로 다른 정보를 줌.

8. 지역활력 (regional_vitality) ★

의미: 데이터센터 유치가 지역 활성화에 미치는 기여도. 인구감소지역 = 데이터센터로 일자리·세수 효과 큼.

방향: 높을수록 좋음. 가중치: 18% — 14회 공모전 주제 직격.

산출:

  • 행안부 2021-11 지정 89개 인구감소지역 공식 리스트
  • 본 109개 시군 데이터셋과 교차 매칭 → 44개 인구감소지역 식별
  • 지정 = 100점, 비지정 군 = 55, 중소시 = 35, 주요도시 = 15, 광역시 = 5 (단계적 차등)

9. 재생에너지 (renewable_access)

의미: RE100 대응 + ESG. 데이터센터 빅테크들의 재생에너지 PPA 요구.

방향: 높을수록 좋음. 가중치: 9%.

산출:

  • 공공데이터포털 전국태양광발전소전기사업허가정보표준데이터 50,000 발전소 raw
  • 정상가동 47,705개만 카운트
  • 제공기관명 ("전북특별자치도 익산시")에서 시군 추출
  • 점수 = log1p(count)×0.5 + log1p(capa_kw/100)×0.5
  • 109 시군 min-max 정규화
