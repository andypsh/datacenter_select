04 파생변수 산출 (Derived Variables)

본 연구의 핵심 기여 — V2 4개 신규 지표를 원본 raw 데이터에서 직접 산출. 산식이 03_분석/derive_indicators.py 에 공개되어 재현 가능.

⚡ power

전력 공급 안정성 — 광케이블 백본(=통신·전력 콜로케이션 프록시)에 인접할수록 송배전 인프라 양호.
count(fiber within 30km) → Min-Max

🏭 industry

산업 클러스터 — 기존 데이터센터 인접 권역에 인력·전력·통신 인프라가 이미 갖춰져 있음.
count(DC within 30km) → Min-Max

🌱 vitality

지역활력 기여도 — DC 결핍 지역 + 인구감소지역에 데이터센터 입주 시 활력 회복 잠재력 큼.
(1 − DC밀도) × 0.6 + 인구감소 × 0.4

☀️ renewable

재생E 접근성 — 호남(태양광)·강원(풍력) 허브 거리. 가까울수록 RE100 / PPA 유리.
exp(−d / 100km) → Min-Max

05 스코어링 방법론

정규화 (Min-Max)

for factor in factors:
    values = [r.factors[factor.key] for r in regions]
    normalized = (values - min) / (max - min)
    if factor.direction == "lower_is_better":
        normalized = 1 - normalized

가중합 스코어링

총_가중치 = Σ w_k
종합점수_i = Σ (w_k × normalized_k_i) / 총_가중치

사용자가 슬라이더로 w_k 를 조절하면 Pinia의 computed 체인이 정규화·점수·랭킹·정책제언 전부 즉시 재계산.

08 정책 제언 5종

대시보드 정책 탭에서 현재 가중치 기준으로 자동 생성되는 제언.

1

수도권 데이터센터 신규 허가 총량제

기본 가중치에서도 Top 10 중 수도권 50%. 송전망 부담 완화 위해 권역별 총량제 + 비수도권 후보지(음성·괴산) 우선 인허가.
대상: 산업통상자원부 · 국토교통부
2

한전 송전망 확충 우선순위 = 비수도권 Top 후보지

대상: 한국전력공사 · 전력거래소
3

인구감소지역 데이터센터 입주 인센티브 패키지

비수도권 Top 10 중 9개가 인구감소지역(=즉시 적용 대상). 법인세 3년 100% 감면 + 전기요금 산업용(을) + 토지비 50% 보조.
대상: 행정안전부 · 기획재정부 · 광역 지자체
4

RE100 대응 — 재생에너지 인접 데이터센터 특구

지역균형 Top 5가 모두 호남(신안·정읍·서천·부안·장성). RE100 데이터센터 특구 + 직접 PPA 허용.
대상: 산업통상자원부 · 한국에너지공단

데이터센터 폐열 활용 의무화

MW급 데이터센터는 폐열의 50% 이상을 지역난방·스마트팜으로 환원 의무화. 지역활력 + 탄소중립 동시 달성.
