extra_data.py

실데이터화 이전엔 5개 신규 지표의 proxy를 다 담았으나, 현재는 2개로 축소:

  1. REGION_PROVINCE: 시군 → 광역도 매핑 (광역시 자치구 합산용)
  2. POPULATION_DECLINE: 행안부 89개 중 본 데이터셋 44개 매칭 set
  3. regional_vitality(name): 인구감소 = 100, 그 외 tier별 차등 (5~55)
def regional_vitality(name: str) -> float:
    if name in POPULATION_DECLINE:
        return 100.0
    if tier(name) == 4:    return 55.0   # 군 단위
    if tier(name) == 3:    return 35.0   # 중소시
    if tier(name) == 2:    return 15.0   # 주요도시
    return 5.0                            # 광역시
