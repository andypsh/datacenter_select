KEA 데이터 (Chrome MCP로 자동 수집)

전국태양광발전소전기사업허가정보표준데이터

  • 출처: 공공데이터포털 표준데이터 (data ID 15107742)
  • 제공기관: 산업통상자원부 산하 + 지자체 종합
  • 크기: 8.2 MB CSV (cp949 인코딩)
  • 레코드: 50,000개 태양광 발전소 — 정상가동 47,705 / 가동중단 1,830 / 폐기 465
  • 주요 컬럼: 태양광발전시설명, 소재지도로명/지번주소, 위도, 경도, 설비용량(kW), 가동상태구분명, 설치연도, 허가일자, 허가기관, 제공기관명

수집 절차 (재현 가이드)

  1. data.go.kr 검색창 → "신재생에너지 발전소"
  2. 표준데이터셋 영역에서 "전국태양광발전소전기사업허가정보표준데이터" 클릭
  3. 상세 페이지 우측 다운로드 영역에서 CSV 버튼
  4. 받은 파일을 02_데이터/raw/KEA_신재생/에 저장
  5. python build_scores.py 재실행
Chrome MCP의 fn_fileDataDown 또는 표준데이터 CSV 버튼 직접 클릭 둘 다 동작 확인 완료.
