알려진 한계 / 보고서 명시 사항

1. 광케이블 데이터 부분 한계
V1 광케이블.csv는 66 노드 (KT 주요 거점)만 포함. 전국 백본망 전체는 아님. 통신 인프라 점수는 "주요 노드 인접도 proxy" 한계 명시 필요.
2. 지역활력 binary 한계
인구감소지역 = 100점 / 비지정 = 단계적 차등은 단순화. 보고서에선 통계청 GRDP 결합 또는 인구 변화율 연속값으로 보강 가능성 제시.
3. IT 인력 추정의 한계
대=1500, 중견=250, 중소=18은 산업 평균. 시군별 실제 종사자는 통계청 KOSIS API 키 발급 후 교체 가능.
4. 좌표 정확성
coords.py의 109 시군 좌표는 시청·군청 위치 근사. 행정구역 centroid가 더 정확하지만 통신 인접도 등 계산 결과에 큰 영향 없음 (km 단위 차이).
5. 109 시군 데이터셋 범위
V1 가중치 데이터셋의 109개로 한정. 제주·울릉 등 일부 시군 미포함. 본 분석에선 전국 약 250 시군구의 대표 109개로 다룸 — 확장 시 모든 시군 포함하려면 V1 추가 raw 수집 필요.
