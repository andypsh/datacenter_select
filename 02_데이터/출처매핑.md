## 데이터 활용 전략

### ✅ 메인 데이터 (산업부 산하 — 차별화 포인트)
1. **KEPCO 전력사용량 & 변전소** — V1에 이미 사용, 최신화 필요
2. **KPX 전력계통/송전망** — 신규 추가 (가장 강력한 차별화)
3. **KICOX 산업단지** — 신규 추가
4. **KEA 신재생 발전소** — 신규 추가 (RE100 / ESG 키워드)

### ➕ 보완 데이터 (외부)
- 기상청 / 국토부 / 통계청 데이터는 보완 용도
- 보고서에서 "산업부 데이터 X종 + 보완 데이터 Y종" 구조로 명시

### Proxy → 실데이터 전환 시점
- **P0**: 출품 전 필수 (3개) — 전력·지역활력·재생에너지
- **P1**: 출품 전 권장 (3개) — 산업·통신·인력
- **P2**: 최신화 (3개) — 자연재해·냉각·토지비

### Proxy 한계 명시 (보고서 작성 시 주의)
- `extra_data.py`의 tier 기반 점수는 합리적 추정이지만 실제 측정값이 아님.
- 보고서에는 "1차 모델은 행정구역 규모 기반 proxy, 본선 모델은 KEPCO·KPX·KEA 실API 활용" 형태로 단계 구분 명시.
- 가중치 슬라이더 + Top 후보지 도출 로직은 데이터 교체 시에도 그대로 유지.
