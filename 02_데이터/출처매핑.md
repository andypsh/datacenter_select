# 데이터 출처 매핑표

> **공모전 필수 조건**: 산업통상자원부 + 산하 공공기관 데이터 활용
> 심사 시 "산업부 데이터를 실제로 활용했는가" 가 첫 번째 체크포인트.

## 데이터 활용 전략

### ✅ 메인 데이터 (산업부 산하 — 차별화 포인트)
1. **KEPCO 전력사용량 & 변전소** — V1에 이미 사용, 최신화 필요
2. **KPX 전력계통/송전망** — 신규 추가 (가장 강력한 차별화)
3. **KICOX 산업단지** — 신규 추가
4. **KEA 신재생 발전소** — 신규 추가 (RE100 / ESG 키워드)

### ➕ 보완 데이터 (외부)
- 기상청 / 국토부 / 통계청 데이터는 보완 용도
- 보고서에서 "산업부 데이터 X종 + 보완 데이터 Y종" 구조로 명시
